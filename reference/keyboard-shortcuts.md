# Keyboard Shortcuts

Speed up your workflow with these keyboard shortcuts in Calcify.

## File Operations

| Shortcut           | Action    | Description                   |
| ------------------ | --------- | ----------------------------- |
| `Ctrl + N`         | New File  | Create a new calculation file |
| `Ctrl + O`         | Open File | Open an existing file         |
| `Ctrl + S`         | Save      | Save current file             |
| `Ctrl + Shift + S` | Save As   | Save with a new name/location |
| `Alt + F4`         | Exit      | Close Calcify                 |
| `Alt + Arrow up`   | Move up   | Move the line one line up     |
| `Alt + Arrow down` | Move down | Move the line one line down   |

## Editing

| Shortcut           | Action     | Description               |
| ------------------ | ---------- | ------------------------- |
| `Ctrl + Z`         | Undo       | Undo last action          |
| `Ctrl + Y`         | Redo       | Redo last undone action   |
| `Ctrl + Shift + Z` | Redo       | Alternative redo shortcut |
| `Ctrl + X`         | Cut        | Cut selected text         |
| `Ctrl + C`         | Copy       | Copy selected text        |
| `Ctrl + V`         | Paste      | Paste from clipboard      |
| `Ctrl + A`         | Select All | Select all text           |

## Text Selection

| Shortcut              | Action                   | Description                      |
| --------------------- | ------------------------ | -------------------------------- |
| `Shift + Arrow Keys`  | Select Text              | Select text in direction         |
| `Ctrl + Shift + ←/→`  | Select Word              | Select word left/right           |
| `Shift + Home`        | Select to Line Start     | Select from cursor to line start |
| `Shift + End`         | Select to Line End       | Select from cursor to line end   |
| `Ctrl + Shift + Home` | Select to Document Start | Select from cursor to start      |
| `Ctrl + Shift + End`  | Select to Document End   | Select from cursor to end        |

## Navigation

| Shortcut      | Action         | Description                |
| ------------- | -------------- | -------------------------- |
| `Arrow Keys`  | Move Cursor    | Move one character/line    |
| `Ctrl + ←/→`  | Move by Word   | Jump to previous/next word |
| `Home`        | Line Start     | Move to start of line      |
| `End`         | Line End       | Move to end of line        |
| `Ctrl + Home` | Document Start | Move to start of document  |
| `Ctrl + End`  | Document End   | Move to end of document    |
| `Page Up`     | Page Up        | Scroll up one page         |
| `Page Down`   | Page Down      | Scroll down one page       |

## Search and Replace

| Shortcut           | Action        | Description                  |
| ------------------ | ------------- | ---------------------------- |
| `Ctrl + F`         | Find          | Open find dialog             |
| `F3`               | Find Next     | Find next occurrence         |
| `Shift + F3`       | Find Previous | Find previous occurrence     |
| `Ctrl + H`         | Replace       | Open find and replace dialog |
| `Ctrl + Shift + F` | Find in Files | Search across multiple files |
| `Escape`           | Close Find    | Close find/replace dialog    |

## Line Operations

| Shortcut           | Action         | Description                     |
| ------------------ | -------------- | ------------------------------- |
| `Ctrl + D`         | Duplicate Line | Duplicate current line          |
| `Ctrl + L`         | Delete Line    | Delete current line             |
| `Ctrl + Shift + K` | Delete Line    | Alternative delete line         |
| `Enter`            | New Line       | Insert new line                 |
| `Shift + Enter`    | New Line Below | Insert line below, cursor stays |
| `Ctrl + Enter`     | New Line Above | Insert line above               |
| `Ctrl + J`         | Join Lines     | Join current line with next     |

## Platform-Specific Notes

### Windows

- Use `Ctrl` for most shortcuts
- `Alt + F4` closes the application
- Windows key combinations may conflict

### Accessibility

- All functions accessible via keyboard
- Screen reader compatible
- High contrast theme support
- Shortcuts work with sticky keys
