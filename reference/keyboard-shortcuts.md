# Keyboard Shortcuts

Speed up your workflow with these keyboard shortcuts in Calcify.

## File Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl + N` | New File | Create a new calculation file |
| `Ctrl + O` | Open File | Open an existing file |
| `Ctrl + S` | Save | Save current file |
| `Ctrl + Shift + S` | Save As | Save with a new name/location |
| `Ctrl + W` | Close File | Close current file |
| `Alt + F4` | Exit | Close Calcify |

## Custom Shortcuts

You can customize shortcuts in **Settings** → **Keyboard Shortcuts**.

### How to Customize

1. Press `Ctrl + ,` to open Settings
2. Go to **Keyboard Shortcuts**
3. Click on the shortcut you want to change
4. Press the new key combination
5. Click **Save**

### Shortcut Conflicts

If a shortcut conflicts with Windows or another application:
- Calcify shortcuts take precedence when focused
- You can disable conflicting shortcuts
- Use alternative shortcuts when available

## Platform-Specific Notes

### Windows
- Use `Ctrl` for most shortcuts
- `Alt + F4` closes the application
- Windows key combinations may conflict

### Accessibility
- All functions accessible via keyboard
- Screen reader compatible
- High contrast theme support
- Shortcuts work with sticky keys
